[distraction.today](https://distraction.today) is a simple quote display web app. It serves up quotes to whoever visits.

The code lives at [icco/distraction.today](https://github.com/icco/distraction.today) and is written in Go with HTML templates.

It's semi-abandoned. It was originally the inspiration to make a book of quotes for a year. At some point I'd like to get back to it.
